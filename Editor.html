# 编辑器

工具栏配置
toolbarKeys重新配置工具栏，显示哪些菜单，以及菜单的排序、分组
excludeKeys如果仅仅想排除掉某些菜单，其他都保留，可以使用 excludeKeys 来配置
modalAppendToBody将菜单弹出的 modal 添加到 body 下，并自定义 modal 的定位和其他样式
编辑器配置
placeholder配置编辑器 placeholder
readOnly配置编辑器是否只读，默认为 false
MENU_CONF
编辑器api
getConfig可查看当前的 hoverbarKeys,excludeKeys等
getAllMenuKeys 查询编辑器注册的所有菜单 key（可能有的不在工具栏上）
getMenuConfig查看菜单的当前配置
getHtml获取非格式化的 html
getText获取当前编辑器的纯文本内容
getSelectionText获取选中的文本
getElemsByType通过 type 获取编辑器的 element 列表
getElemsByTypePrefix通过 type 前缀获取编辑器的 element 列表
getFragment获取选中的内容，json 格式
setHtml重置编辑器的 HTML 内容。【注意】只能解析 editor.getHtml() 返回的 HTML 格式，不支持自定义 HTML 格式
dangerouslyInsertHtml如果是 editor.getHtml() 获取的 HTML 格式，可以完美解析
insertText在选区插入文本
insertBreak在选区回车换行
deleteBackward向后删除，相当于按 backspace 键
deleteForward向后删除，相当于按 delete 键（部分键盘没有这个键）
deleteFragment删除选中的内容
getParentNode获取一个节点的父节点
toDOMNode获取一个节点对应的 DOM 节点
insertNode在选区插入一个节点
insertNodes在选区插入多个节点
removeNodes删除选区所在的节点
marks获取选中文字的标记（文本样式）
addMark为选中的文本添加标记（文本样式）
removeMark对选中的文字，取消标记（文本样式）
id 属性获取编辑器 id
fullScreen设置为全屏
unFullScreen取消全屏
isFullScreen编辑器是否全屏
focus聚焦到编辑器
blur失焦编辑器
isFocused判断当前编辑器是否聚焦
enable判断当前编辑器是否只读
disable取消禁用，取消只读
isDisabled判断当前编辑器是否只读
clear清空编辑器内容
updateView强制更新视图
scrollToElem滚动到指定元素
isEmpty判断当前编辑器内容是否为空（只有一个空段落）
isText判断一个节点是否是 text
isElement判断一个节点是否是 elem
isFullScreen编辑器是否全屏
isDisabled判断当前编辑器是否只读
isSelectedAll判断编辑器是否全部选中
select选中一个指定的选区
selectAll选中所有内容
insertText在选区插入文本
insertBreak在选区回车换行
insertNode在选区插入一个节点
insertNodes在选区插入多个节点
deselect取消选中
move移动光标
moveReverse反向移动光标
showProgressBar显示进度条，一般用于上传功能
hidePanelOrModal隐藏当前的弹框 （如插入链接） 和下拉列表（如设置标题、设置字体）
getSelectionPosition获取选区的定位，将视情况返回 left right top bottom 的其中几个
getNodePosition获取某个节点的定位，将视情况返回 left right top bottom 的其中几个
destroy销毁编辑器和工具栏
event
onMaxLength当达到 maxlength 限制时，触发该回调函数
onCreated编辑器创建完毕时的回调函数
onChange编辑器内容、选区变化时的回调函数
onDestroyed编辑器销毁时的回调函数。调用 editor.destroy()即可销毁编辑器
onFocus编辑器 focus 时的回调函数
onBlur编辑器 blur 时的回调函数
customPaste自定义粘贴。可阻止编辑器的默认粘贴，实现自己的粘贴逻辑
fullScreen
unFullScreen
fullScreen
scroll
modalOrPanelShow
modalOrPanelHide

slate 官网链接(Slate (opens new window)是一个 完全 可定制的富文本编辑器框架)open in new window
